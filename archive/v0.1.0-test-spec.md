- [ ] Sample 20 completed games with box scores: verify winning pitcher has IP > 0
- [ ] Sample 10 completed games: verify each has notable_events JSON array (may be empty)

- [ ] GET /api/standings returns 20 rows

- [ ] POST /api/sim/speed with body `{"speed":"paused"}` stops sim
- [ ] No new games appear in game_log while paused
