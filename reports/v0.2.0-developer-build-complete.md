## Architecture Decisions / Rulings Applied
- **AB-01**: Two game counters (teams.games_played per-team, leagues.current_game_number league-wide)
- **AB-04**: DFA immediately vacates 40-man, team_id retained during window
- **AB-05**: SERVICE_YEAR_GAMES=30, FREE_AGENT_SERVICE_GAMES=180
- **AB-08**: Spring cuts atomic with spring_cuts_done_season
- **AB-10**: In-season prospect dev only for age 18-25 AA/A/Rookie; bust at 26
- **AB-12**: Forced minimum 3 trades after deadline marker
- **AB-14**: Spring cuts go straight to FA (not waivers)
- **AB-18**: rosterMaintenance runs as sibling transaction after simulateGame, every tick
- **CB-07**: Waiver claim resolution in ONE db.transaction(), idempotent WHERE guard
- **CB-08**: Service time additive-only, no subtraction path
